# 履修登録（時間割）データ設定マニュアル

プレイヤーごとの時間割（履修科目）をWebサイト上に表示させるためのマニュアルです。

## 1. ファイルの作成場所
`Nilatorf/ClassSchedule/` フォルダ内に、各プレイヤーのIDに対応したテキストファイルを作成します。
例： `PL1.txt`, `PL2.txt`

## 2. 記述形式
1行に1科目をカンマ区切り（CSV形式）で記述します。

**フォーマット:**
`曜日,時限,科目名`

- 曜日: `月`, `火`, `水`, `木`, `金`, `土`, `日` のいずれか1文字。
- 時限: `0` 〜 `6` などの数字。（`時間割.txt` で定義されている時限と一致させること）
- 科目名: 任意のテキスト（例: 現代文読解、力学基礎）

**記述例 (`PL1.txt`):**
```text
月,1,現代文読解
月,2,力学
火,1,古文
火,3,化学
水,2,生物
木,1,数学
金,4,英語
```

## 3. 注意事項
- 文字コードは必ず **UTF-8** で保存してください。
- 曜日や数字の前後に余分なスペースを入れないでください。
- プレイヤーIDに対応するファイル（例: `PL1.txt`）が見つからない場合は、時間割表は空欄（枠のみ）で表示されます。
